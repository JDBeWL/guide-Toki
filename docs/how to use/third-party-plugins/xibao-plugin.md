# 喜报/悲报

插件仓库地址：[@ifrank/xibao](https://github.com/ifrvn/cecilia/tree/main/plugins/xibao)

作者GitHub个人页：[ifrvn](https://github.com/ifrvn)

```
<!-- 指令 -->

    喜报 <text...>
    喜报 要在喜报上写的内容，支持换行
<!-- 功能 -->
    生成一张喜报

    悲报 <text...>
    悲报 要在悲报上写的内容，支持换行
<!-- 功能 -->
    生成一张悲报

```

**使用示例**
```
    喜报 这可以喜
    悲报 这不可以喜
```
